# SkyCodeHub

Coding assessment and exam platform for colleges — secure online tests, real-time code execution, and proctored exams via Safe Exam Browser (SEB).

## What we build

- **Dashboard** — Web platform for admins, teachers, and students to create, manage, and take coding & MCQ tests.
- **Backend API** — Node.js/Express service handling auth, test management, analytics, and submissions.
- **Code Judger** — Sandboxed code execution engine for running and grading student submissions.
- **SEB Integration** — Locked-down exam browser for proctored, anti-cheat assessments.
